# @rnw-community/react-native-payments-example

Private. Bare React Native example app demonstrating the Payment Request API.

## Overview

- Imports `App` from `react-native-payments-example-lib`
- Traditional React Native setup with `ios/` and `android/` native directories
- Entry point: `index.js` → `AppRegistry.registerComponent`

## Dependencies

`react-native-payments-example-lib` (workspace), `react`, `react-native`.

This is an example/demo package — no tests, no build, no publish.
