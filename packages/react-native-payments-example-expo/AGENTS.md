# @rnw-community/react-native-payments-example-expo

Private. Expo-based example app demonstrating the Payment Request API with Expo Router and web support.

## Overview

- Uses Expo Router for navigation (`app/_layout.tsx`)
- Imports `App` from `react-native-payments-example-lib`
- Supports iOS, Android, and Web platforms
- Expo config plugin sets up merchant identifiers for Apple Pay / Google Pay

## Dependencies

`@rnw-community/react-native-payments` (workspace), `react-native-payments-example-lib` (workspace), `expo`, `expo-router`, `react`, `react-native`, `react-native-web`.
